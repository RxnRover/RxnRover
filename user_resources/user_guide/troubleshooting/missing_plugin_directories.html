My Plugin Directories are Missing!

Issue

One or more of the Plugin Directories are missing.

Cause

One or more Plugin Directories have been deleted, or Rxn Rover has not generated the directories yet.

Solutions

Rxn Rover generates the Plugin Directories when it is run for the first time, and recreates any missing plugin directories each time it is run. Try running Rxn Rover without an experiment to regenerate the plugin directories without the extra steps of creating a dummy experiment.
